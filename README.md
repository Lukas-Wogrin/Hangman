# Hangman – C# Windows Forms
A Hangman game built with C# and Windows Forms.
A random word is chosen from a list of 10 words — guess it letter by letter before the hangman is complete!
# Features
10 words randomly selected each game
Letters displayed as ____ until guessed
Visual hangman drawing step by step
Win/lose detection
# Technologies
C# (.NET)
Windows Forms (WinForms)
Visual Studio
# How to Run
Clone the repository:
Bash
git clone https://github.com/Lukas-Wogrin/hangman.git
Open Hangman.sln in Visual Studio
Press F5 to run
